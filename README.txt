How to run this project?

1. Copy the project folder to driver D: (e.g. D:\SentimentTM).

2. Go to each modules, from 1.1 to 3.1, read the README.txt file and
   run each module of the project in Python 2.7 64-bit.

3. If there are some missing packages for Python 2.7 64-bit, try to
   install them through conda or pip. If it isn't successful go to
   folder D:\SentimentTM\Requirements\Python27_64bit_packages and
   install them manually.
